# Kali Linux Installation

## Objective

Install Kali Linux inside VMware Workstation to create a secure and isolated environment for cybersecurity learning.

---

## Lab Setup

| Component | Details |
|-----------|----------|
| Host OS | Windows |
| Hypervisor | VMware Workstation |
| Guest OS | Kali Linux |

---

## Software Used

- VMware Workstation
- Kali Linux ISO

---

## Procedure

1. Downloaded Kali Linux ISO.
2. Created a new virtual machine.
3. Allocated RAM, CPU, and Storage.
4. Installed Kali Linux.
5. Verified successful boot.
6. Performed initial configuration.

---

## Result

Successfully installed Kali Linux inside VMware.
