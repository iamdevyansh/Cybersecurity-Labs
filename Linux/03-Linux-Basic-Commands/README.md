# Linux Basic Commands

## Objective

Learn and practice the fundamental Linux commands used in daily system administration and cybersecurity.

---

## Commands Practiced

| Command | Description |
|----------|-------------|
| pwd | Show current directory |
| ls | List files |
| cd | Change directory |
| mkdir | Create directory |
| touch | Create file |
| cp | Copy files |
| mv | Move files |
| rm | Delete files |
| clear | Clear terminal |
| history | Show command history |
| whoami | Show current user |
| cat | View file contents |
