# 🐧 Linux Labs

## Overview

This directory contains my hands-on Linux learning journey as part of my cybersecurity roadmap.

Linux is one of the most essential operating systems in cybersecurity. Through these labs, I am building practical skills required for ethical hacking, security operations (SOC), cloud computing, and system administration.

All activities are performed inside my personal home lab using virtual machines.

- Learn Linux fundamentals
- Understand the Linux file system
- Practice command-line operations
- Configure a cybersecurity home lab
